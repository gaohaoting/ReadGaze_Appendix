# UIST 2026 补充材料与匿名 — 核对清单

本清单依据 [UIST 2026 CFP](https://uist.acm.org/2026/cfp/) 与 [Author Guide](https://uist.acm.org/2026/author-guide/) 整理，投稿前请逐项确认。

## 截止时间与渠道

| 项目 | 说明 |
|------|------|
| 论文 PDF | 2026-03-31 AoE（以官网为准） |
| 材料（视频、补充材料、无障碍 alt-text、字幕等） | **2026-04-09 AoE** |
| 提交平台 | [PCS](https://new.precisionconference.com/submissions)（SIGCHI / UIST 2026 / Papers） |
| 总文件大小 | 全部上传文件 **&lt; 300 MB** |

## 匿名（双盲）

- [ ] 论文 PDF 中无作者姓名、单位、致谢等可识别信息。
- [ ] 视频与字幕匿名。
- [ ] **所有 supplementary material**（含本仓库若被引用）已去标识；未匿名可能被 **desk reject**。
- [ ] 文中若给出代码/数据链接，须为**匿名化 URL**（例如通过 anonymous.4open.science），且链接目标页面与 README 仍无身份信息。
- [ ] 引用自己既往工作时使用第三人称，并附完整参考文献（见 Author Guide）。

## 评审独立性

- [ ] 正文论证**不依赖**审稿人必须访问未在 PCS 提交的外部材料；附录/开源为 **optional**，不应成为评审必要条件（CFP 明确说明）。

## Rebuttal（与初次投稿区分）

- [ ] Rebuttal 中**一般不允许外链**，除非 AC 明确要求（Author Guide）。

## 无障碍（若适用）

- [ ] 图表 alt-text、视频字幕等在 **2026-04-09** 前按 CFP 要求提交。
